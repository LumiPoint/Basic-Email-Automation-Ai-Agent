##  Project Overview
This project is an intelligent email management system built using **Zapier** and **OpenAI**. It is designed to analyze incoming emails, automatically categorize them based on priority (Support, Sales, General), and generate context-aware draft replies to enhance productivity.

## Key Features
- **Smart Categorization:** Automatically sorts emails using AI analysis.
- **Auto-Drafting:** Generates professional draft responses using GPT models.
- **Workflow Integration:** Seamlessly connects Gmail/Outlook with Slack or CRM tools via Zapier.

## Tech Stack
- **Automation:** Zapier
- **AI Model:** OpenAI (GPT-3.5/4)
- **Email Service:** Gmail API / Microsoft Graph API
